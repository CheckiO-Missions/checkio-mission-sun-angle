Zacznijmy od przygotowania naszej linii wejściowej przed obliczeniami. Przekażemy do zmiennych "h" i "m" odpowiednio wartości godzin i minut, już przekonwertowane na typ danych całkowitych za pomocą funkcji funkcji map() i int(). Forma zapisu przez przecinek "h, m = map(int, ...)" jest równoważna następnej formie "h = int(...)" oraz "m = int(...)". Używamy funkcji int() do każdego elementu podzielonego wiersza używając separatora ":". 
time = "07:00"
h, m = map(int, time.split(':'))

def sun_angle(time: str) -> Union[float, str]:
    h, m = map(int, time.split(':'))
    angle = 15 * h + m * 0.25 - 90
    return angle if 0 <= angle <= 180 else "I don't see the sun!"

Proszę spojrzeć na to rozwiązanie. Z pomocą modułu datetime możemy wykonać nasze obliczenia w jednostkach czasu, a dopiero potem przetłumaczyć wartość na jednostki kątowe. 
import datetime
def sun_angle(time):
    start=datetime.datetime.strptime("06:00","%H:%M")
    end=datetime.datetime.strptime("18:00","%H:%M")
    time=datetime.datetime.strptime(time,"%H:%M")
    diff=(time-start).seconds/60*0.25
    return diff if start<=time<=end else "I don't see the sun!"
