每个真正的旅行者都必须知道如何做三件事：灭火、找水和从周围的大自然中提取有用的信息。编程不会帮您灭火和找水，但在提取信息方面，它可能正是您所需要的。

您的任务是根据一天中的时间找出太阳在地平线上的角度。输入数据：早上6:00太阳从东方升起，对应角度为0度。中午12:00太阳到达天顶，这意味着角度等于90度。下午 6:00 是日落时间，因此角度为 180 度。如果输入的是晚上的时间（早上6:00之前或下午6:00之后），您的函数将返回 "我看不到太阳！"。

输入： 一天中的时间。

输出： 太阳角度，四舍五入到小数点后2位。
